## Desktop Composer 是什么？

**Desktop Composer** 是一款 macOS 外观配置文件管理工具，开发商为 Apptorium。它的核心逻辑是：你提前把"壁纸 + 深色/浅色模式 + Dock 设置 + 终端/VS Code/Obsidian 等应用的配色主题"打包成一个「配置文件」，之后一键切换，所有设置同时生效。

换句话说——你不是在改壁纸，你是在切换整个桌面的"工作状态"。

Mac 原生系统当然也支持这些设置，但每一项都得单独点。Desktop Composer 的价值在于：把你平时要点十次才能完成的事，压缩成一次点击。

## 为什么说 Mac 主题切换这件事值得专门装一个软件？

很多人没意识到自己切换桌面状态的频率有多高。

写代码的时候，你想要深色背景、最小化 Dock、隐藏桌面图标。打开 Zoom 开会的时候，你希望桌面干净、壁纸专业、侧边栏 Widget 消失。到了傍晚，你想把系统切回浅色模式，Terminal 换个暖调配色。

每次都手动点，每次都是打断注意力的小事。积累起来，烦。

macOS 内置没有"配置文件"概念，不同状态的外观设置没办法一键保存和调用。这是 Desktop Composer 存在的理由。

方式有三种：

- 点菜单栏图标，选择配置文件

另外，Desktop Composer 支持接入 **Apple 快捷指令** 和 **Focus 专注模式**，可以设置"进入专注模式时自动切换到对应桌面配置"。
